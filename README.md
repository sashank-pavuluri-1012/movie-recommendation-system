# Movie Recommendation System 🎬

## 📌 Project Overview
This project recommends similar movies based on content using machine learning techniques.

## 🚀 Features
- Select a movie from dropdown
- Get top 5 similar movies
- Interactive web app using Streamlit

## 🛠 Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit

## ⚙️ How It Works
- Data preprocessing and feature engineering
- Combined features: overview, genres, keywords, cast
- Converted text into vectors using CountVectorizer
- Used cosine similarity to recommend movies

## ▶️ Run the Project
pip install streamlit scikit-learn pandas  
streamlit run app.py
